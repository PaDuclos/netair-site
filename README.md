# Netair — Site vitrine

Site web vitrine de Netair, fabricant de filtres pour systèmes de ventilation générale (CTA).

## Structure du projet

```
netair-site/
├── index.html              — Page d'accueil
├── catalogue.html          — Catalogue produits
├── assets/                 — Logo, favicon et ressources graphiques
└── fiches-techniques/
    ├── generateur_fiches.html         — Générateur de fiches techniques (17 produits)
    └── Générateur fiches techniques — Netair.pdf  — Export PDF du générateur
```

## Fiches techniques

Le générateur de fiches techniques (`fiches-techniques/generateur_fiches.html`) permet de :
- Visualiser les fiches des 17 références Netair
- Exporter chaque fiche en PDF via impression
- Sauvegarder / charger les données au format JSON

## Gamme produits

Filtres conformes double classification **EN779** et **EN16890 (ISO)** :

| Famille | Références |
|---|---|
| Préfiltres & filtres plans | NETMETAL, NETFIL, NETFIBRE, NETPLY, NETPLAN |
| Filtres à poches souples | NETBAG S |
| Filtres compacts / poches rigides | NETPAK S (BORA, CILIA, AZUR, LUMEN, DUO) |
| Filtres HEPA / T.H.E | NETCEL V (AZUR, NIVAL), NETPAK V LAM |
| Filtres à charbon actif | NETCARB (CILIA, AZUR, NIVAL, BAG) |
